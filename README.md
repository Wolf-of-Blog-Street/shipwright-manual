# Shipwright Manual

Public inspection copy of the Shipwright Manual.

Open the [complete visual manual](docs/viewers/ship-manual.html). Its Manual Library contains the whole-ship
manual, Captain inspection volumes, the shared officer standard, all ten department books, and the evidence-backed
build view.

This is a deliberately scoped publication package. It does not publish the private Shipwright repository, Brain
records, credentials, research transcripts, or runtime state.
